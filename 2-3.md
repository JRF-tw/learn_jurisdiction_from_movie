# 教學區

## 教學目標

1. 探究法律的本質與精神
2. 培養法律意識
3. 提升法律知能
4. 落實法律實踐
5. 培養人文素養，獨立思考能力，創造多元智慧

## 教材分析

就本片的教學重點，有三方面可加以著墨探討，首先是法律精神的涵泳，在接受法律常識、法律會考眾多方面認知的洗禮之後，學生究竟能否掌握法律的精神其實根植於公平、正義的社會環境中，在知法、行法和守法的過程中，不只能夠服從並且能夠思辯和建構。其次本片法庭戲頗多，瞭解陪審團制度和懲罰性賠償的意義將有助於瞭解台灣和英美法系，其法律行使之不同。第三是建立對同性戀者的正確認知，學習公平對待並尊重他人。

## 活動方式

1. 活動前
  1. 影片欣賞的要點
    + 編劇（對白）：如何反應。
    + 演員：如何彰顯、表達。
    + 藝術總集：如何強化（音樂、攝影、服裝）。

  2. 給予影片欣賞作業單：

  3. 分組。

2. 活動時

1. 影片欣賞後，小組分享、討論並報告作業單之題目。
2. 教師統整，指導老師講評。
3. 教師簡述議題的概念並引導學生思考相關課題。
4. 小組討論並報告之。
5. 教師統整，指導老師講評。思考→討論→批判→沈澱→行動。


3. 活動資料說明

  1. 作業單

    1. 請簡要說明本片的主旨與內涵
    2. 請分享本片中精彩的對白
    3. 請抒寫觀看本片後的感想與啟發
    4. 從本片中，你覺得法律的本質和基本精神為何？
    5. 本片法庭的審理過程和台灣的法庭有何不同？
    6. 如果你是安德魯，你會不會對事務所提出控告？你所堅持的理由為何？
    7. 如果你是喬，會不會接受這樣的案例？你所抱持的理念為何？
